---
title: "Install Averfo"
---

Installing Averfo on your website only takes a couple of minutes.

## New website?

1. Navigate to the [Websites](https://averfo.com/websites) page on Averfo.
2. If you haven't already, click `Add website` and enter your domain name.
3. Click the copy button under `Script Installation`.
4. Head to your website `<head>` HTML area and paste it just before the closing `</head>` tag. If you need assistance on where this should go, please [get in touch](mailto:support@averfo.com) and we'll be happy to help!

## Already have a website?

1. Navigate to the [Websites](https://averfo.com/websites) page on Averfo.
1. Click the `Settings` button next to the website you wish to install Averfo onto.
1. Click the copy button under `Script Installation`.
4. Head to your website `<head>` HTML area and paste it just before the closing `</head>` tag. If you need assistance on where this should go, please [get in touch](mailto:support@averfo.com) and we'll be happy to help!

## Wordpress Plugin

We have an official WordPress plugin to make the install process easier. The plugin is currently being submitted to the plugin marketplace, but you can download an install manually here in the meantime:

1. [Download the Beta plugin](https://averfo.com/storage/wordpress/averfo.zip) and install the zip via your WordPress dashboard.
1. Activate the plugin and you are done, simple!
1. Check back later on the Averfo dashboard to see your new stats flow in.
